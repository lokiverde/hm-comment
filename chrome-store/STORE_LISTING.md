# HM Comment - Chrome Web Store Listing

## Extension Name
HM Comment - AI LinkedIn Comment Generator

## Short Description (132 chars max)
Generate professional AI-powered LinkedIn comments instantly. Multiple tones. One click. Boost your engagement 10x.

## Detailed Description

**Stop staring at blank comment boxes. Start engaging like a pro.**

HM Comment adds a sleek "HM" button to every LinkedIn post in your feed. One click generates 3 AI-powered comment options tailored to each post's content.

### 🚀 Features

**Multiple Comment Styles**
- Short & punchy (2-3 sentences)
- Medium depth (3-4 sentences with insight)  
- Longer thought leadership (4-5 sentences)

**Custom Tones**
- Default Professional
- Hormozi (direct, value-focused)
- Casual (friendly, conversational)
- Professional (formal, business-appropriate)

**Smart AI**
- Powered by Claude AI via secure webhook
- Reads post context for relevant responses
- Never generic — always tailored

**Privacy First**
- Your webhook, your data
- No tracking, no analytics
- Configure your own AI backend

### 💡 Perfect For
- Real estate agents building their brand
- Business owners growing their network
- Sales professionals staying visible
- Anyone who wants to engage more without spending hours writing

### ⚡ How It Works
1. Browse LinkedIn as usual
2. See the "HM" button on every post
3. Click to generate 3 comment options
4. Pick your favorite and post

### 🔧 Setup
1. Install the extension
2. Click the extension icon
3. Enter your webhook URL (provided separately)
4. Start commenting!

## Permissions Justification

**activeTab**: Required to detect when user is on LinkedIn and inject the comment button interface.

**scripting**: Required to inject the "HM" button into LinkedIn post elements and handle user interactions.

**storage**: Required to save user preferences (webhook URL, default tone selection) locally in the browser.

**host_permissions (linkedin.com)**: Required to add the comment generation button to LinkedIn posts. The extension only activates on LinkedIn pages and does not access any other websites.
